# 王子賞タイピング

ド派手なオンライン対戦タイピングゲーム。`index.html` 一枚で完結しています。

## 遊び方

`index.html` をそのままブラウザで開く、もしくは GitHub Pages で公開するだけで動作します。

- **CPU対戦**：3キャラから1人を選び、CPUと1対1で対戦
- **オンライン対戦**：サーバー不要のP2P通信（PeerJS）。部屋を作って発行された合言葉を友達に送るだけ

## キャラクター

| キャラ | タイプ | 頭脳 | 力 | 必殺技 |
|---|---|---|---|---|
| あ | 電光の早打ち | 9 | 4 | 電光石火：相手の次のお題を短縮 |
| い | 不動の盾守 | 6 | 6 | 不屈の心：ミス3回分を無効化 |
| う | 剛拳の一打 | 4 | 9 | 渾身の一撃：次の一打を3倍ダメージ |

合計ポイントを13で統一し、3キャラのバランスを取っています。

## キャラ画像について

`assets/あ.png` `assets/い.png` `assets/う.png` を好きな画像に差し替えるだけでキャラ見た目を変更できます（現在はプレースホルダーの文字アイコンが入っています）。画像が見つからない場合は自動でアイコン表示にフォールバックします。

## 公開方法（GitHub Pages）

1. このフォルダの内容をリポジトリにそのままpush
2. リポジトリの Settings → Pages → Branch を選択して保存
3. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます

## ファイル構成

```
index.html      ... ゲーム本体（HTML/CSS/JS全部入り）
assets/あ.png   ... キャラ「あ」の画像
assets/い.png   ... キャラ「い」の画像
assets/う.png   ... キャラ「う」の画像
```
